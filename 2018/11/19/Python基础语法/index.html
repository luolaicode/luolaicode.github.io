Python基本语法

1.注释

注释就是代码中不参与执行的说明性文字

  • 单行注释：在一行文字前面加#号
  • 多行注释：使用三个单引号或者三个双引号引起来（一般用双引号）

  • 缩进：每行代码前的空白叫做缩进，python中每行代码前缩进的个数是有规范的，不能随便缩进，也不能在该出现缩进的时候缩进。

4.常用的数据类型

  • int 整型(整数) 例如：10,100，-100
  • float 浮点型(小数)，例如：1.2,3.22，-3.14
  • str 字符串，例如：’hello!’,’你好世界’,’123’….
  • bool 布尔，只有Ture 或 False(首字母必须大写)，都是关键字，Ture-真，成立，肯定 False-假，不成立，否定
  • list（列表），dict（字典），set（集合），tuple（元祖），迭代器，生成器，function等。

a.定义：变量是用来在内存中开辟空间保存数据的

  • 语法：变量=数据

  • 规范：见名知义（看到变量名大概知道这个变量中存的什么数据），所有字母都是小写，如果变量名是由多个单词组成，单词与单词之间下划线隔开
