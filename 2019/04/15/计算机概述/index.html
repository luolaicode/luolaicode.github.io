计算机概述与linux

2019-04-15 linux

计算机概述

1.计算机由硬件和软件构成

  • 计算机硬件的五大部件：运算器、控制器、存储器、输入设备、输出设备
  • 计算机软件：系统软件（操作系统、编译程序）和应用软件
  • 操作系统会负责管理和操作硬件，并提供简单的人机交互的接口

linux命令

1.命令书写方式: 命令 命令的查参数 命令作用的对象

2.Linux常用的命令：

  1. w / who / whoami / last - 查看登录用户情况
  2. adduser / passwd - 创建用户修改密码
  3. date / cal - 查看日期日历
  4. write / wall / mesg - 发送消息
  5. clear - 清除屏幕
  6. logout / exit - 退出登录
  7. man / info / –help - 查看帮助
  8. history - 历史命令 - !命名编号 - 再次执行历史命令
  9. reboot - 重启系统 - init 6
  10. shutdown - 关机 - init 0

    3. Linux文件操作相关命令：

  11. pwd - print working directory - 打印工作目录
  12. cd - change directory - 改变目录 - 相对路径和绝对路径
  13. ls - list directory contents - 列出目录下的内容
    -l - 长格式查看
    -a - 查看所有文件 - 在Linux系统中以点开头的文件和文件夹是隐藏的
    -R - 递归式的查看 - 遇到文件夹要递归的列出文件夹下的内容
  14. cat - concatenate - 连接多个文件（查看文件的内容）
  15. touch - 创建空文件或者修改已有文件的最后访问时间
  16. mkdir - make directory - 创建文件夹
    -p - 如果需要就创建父文件夹
  17. rm - remove - 删除
    -f - 强制删除
    -r - 递归删除 - 删文件夹需要指定该参数
  18. rmdir - remove empty directory - 删除空文件夹
  19. wget - 通过网络获取文件
    • 例如：wget https://www.python.org/ftp/python/3.7.3/Python-3.7.3.tar.xz
    • 例如：wget https://www.python.org/ftp/python/3.7.3/Python-3.7.3.tgz
  20. gzip / gunzip - 文件压缩/解压缩（gz/tgz）
    • 例如：gunzip Python-3.7.3.tgz
  21. xz - 文件压缩/解压缩（xz）
    -z - 压缩
    -d - 解压缩
    - 例如：xz -d Python-3.7.3.xz
  22. tar - archive - 文件归档/解归档
    -xvf - 解归档（将一个文件拆成多个文件）
    - 例如：tar -xvf Python-3.7.3.tar
    -cvf - 创建归档文件（将多个文件合并成一个文件）
  23. wc - word count - 查看文件行数、单词数、字符数
    -l - 查看行数
    -w - 查看单词数
    -c - 查看字符数
  24. sort - 文件排序
    -r - 降序（默认是从小到大 - 升序）
  25. uniq - unique - 文件去重（相邻相同的内容会被过滤掉）
  26. head / tail - 查看文件的开头/结尾部分
  27. more / less - 分页查看文件
    • 例子：cat -n taobao.html | more
  28. diff - different - 比较文件的差别
    19.echo — 打印输出某个内容，echo 字符串 > 文件名 –在某个文件里添加字符串（覆盖） echo 字符串 >> 文件名 – 在某个文件内容后面添加字符串(在后面追加)

4.常用命令

Ctrl+D - 结束输入
Ctrl+C - 中断一个正在执行的命令
Ctrl+W - 删除命令中的一块
Ctrl+A - 光标到行首
Ctrl+E - 光标到行尾
